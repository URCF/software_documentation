**URMAP** (Ultra-fast Read Mapper) is ~10x faster than BWA with
comparable accuracy on benchmark tests.[1][2]

Installed Versions
------------------

### Picotte

**URMAP 1.0.1441** is installed on Picotte. Use the modulefile:

`    urmap/git`
