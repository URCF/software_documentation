**Cytoscape** is software for visualizing complex network data and
integrating with attribute data.[1]

Installed Versions
------------------

**Cytoscape 2.7.0** and **3.2.1** are installed on Proteus. The
modulefiles are:

`    cytoscape/2.7.0`
`    cytoscape/3.2.1`

These both use [Java 1.7.0](/Java "wikilink")
