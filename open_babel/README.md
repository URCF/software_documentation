**Open Babel**[1][2] is a chemical toolbox designed to speak the many
languages of chemical data.

Installed Versions
------------------

**Open Babel 3.1.1** compiled with Intel compilers is installed. Use the
modulefile:

`    openbabel/intel/2020/3.1.1`
