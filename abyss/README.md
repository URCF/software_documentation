**ABySS** is Assembly By Short Sequences - a de novo, parallel,
paired-end sequence assembler[1][2]

Installed Versions
------------------

ABySS 2.3.5 is available on Picotte. Modulefile:

-   abyss/2.3.5

This is MPI-enabled, compiled with GCC.
