Proteus has [CRISPR-related software](https://github.com/ctSkennerton)
installed: libcrispr, crass, crisptools.

Installed Versions
------------------

The installed versions are the current development versions from the git
repositories.

To use, load the module:

`    crispr/gcc/1.0`
